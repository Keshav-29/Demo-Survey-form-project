\# 🌈 Survey Form Application (MERN Stack)



A full-stack survey application built using \*\*React, Node.js, Express, and MongoDB Atlas\*\*.  

Users can submit feedback through a modern UI, and responses are stored securely in a cloud database.



\---



\## 🚀 Features



\- 📝 User-friendly survey form

\- 📡 REST API using Express.js

\- ☁️ Cloud database (MongoDB Atlas)

\- 🔄 Real-time data submission

\- 🎨 Modern UI with CSS styling

\- 🔐 Secure environment variables using `.env`



\---



\## 🛠️ Tech Stack



\- \*\*Frontend:\*\* React.js

\- \*\*Backend:\*\* Node.js, Express.js

\- \*\*Database:\*\* MongoDB Atlas

\- \*\*Version Control:\*\* Git \& GitHub
